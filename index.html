📊 Google Sheets Auto-Sync Setup
Lead save गर्दा automatically Google Sheets मा sync हुन्छ। एकपटक Apps Script URL set गरे पुग्छ।
Setup Steps (एकपटक मात्र):
1. sheets.google.com मा नयाँ Sheet बनाउनुस्
2. Extensions → Apps Script खोल्नुस्
3. तलको code paste गर्नुस् र Save गर्नुस्:
function doPost(e){ try{ var d=JSON.parse(e.postData.contents); var ss=SpreadsheetApp.getActiveSpreadsheet(); var sh=ss.getSheetByName('Leads')||ss.insertSheet('Leads'); var hdr=['ID','Name','Phone','District','Municipality','Area','Model','Status','Priority', 'Source','Finance','Exchange','Exch Model','Follow-Up','Visit Date','Assigned To', 'Booking','Booking No','Booking Date','Delivery Date','Actual Delivery', 'Engine No','Chassis No','Bike Reg No','Colour','Sold Amount','Remark','Notes','Created']; if(sh.getLastRow()===0) sh.appendRow(hdr); if(sh.getLastRow()>1) sh.getRange(2,1,sh.getLastRow()-1,hdr.length).clearContent(); d.leads.forEach(function(l){ sh.appendRow([l.id,l.name,l.phone,l.district||'',l.municipality||'',l.address||'', l.interestedModel,l.leadStatus,l.priority,l.source,l.finance||'',l.exchange||'No', l.exch_model||'',l.followUpDate||'',l.visitDate||'',l.assignedTo||'', l.bookingStatus||'No',l.bookingNo||'',l.bookingDate||'',l.deliveryDate||'',l.actualDelivery||'', l.engineNo||'',l.chassisNo||'',l.bikeRegNo||'',l.bikeColour||'',l.soldAmount||'', l.remark||'',(l.notes||[]).length,l.createdAt||'']); }); return ContentService.createTextOutput(JSON.stringify({status:'ok',count:d.leads.length})) .setMimeType(ContentService.MimeType.JSON); }catch(err){ return ContentService.createTextOutput(JSON.stringify({status:'error',msg:err.message})) .setMimeType(ContentService.MimeType.JSON); } } 4. Deploy → New Deployment → Web App
   • Execute as: Me
   • Who has access: Anyone
5. Deploy गरेपछि आउने URL तल paste गर्नुस्

🗂️ Google Drive Auto-Backup Setup
हरेक lead save हुँदा automatically Google Drive मा JSON backup file बन्छ। एकपटक मात्र setup गर्नु पर्छ।
📋 Setup Steps (एकपटक मात्र):
1. script.google.com खोल्नुस् → New Project
2. तलको code सबै select गरेर paste गर्नुस् (पुरानो code हटाउनुस्):
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var folder = DriveApp.getRootFolder(); // "Divya AutoHub Backups" folder बनाउने var folders = folder.getFoldersByName('Divya AutoHub Backups'); var backupFolder = folders.hasNext() ? folders.next() : folder.createFolder('Divya AutoHub Backups'); // Timestamp सहित file नाम var ts = Utilities.formatDate( new Date(), 'Asia/Kathmandu', 'yyyy-MM-dd_HH-mm-ss'); var fname = 'leads_backup_' + ts + '.json'; backupFolder.createFile(fname, JSON.stringify(data, null, 2), MimeType.PLAIN_TEXT); // पुराना backup हटाउने (20 भन्दा बढी भए) var files = backupFolder.getFilesByName; var allFiles = backupFolder.getFiles(); var fileList = []; while (allFiles.hasNext()) fileList.push(allFiles.next()); fileList.sort(function(a,b){ return b.getDateCreated()-a.getDateCreated(); }); if (fileList.length > 20) { for (var i=20;i<fileList.length;i++) fileList[i].setTrashed(true); } return ContentService .createTextOutput(JSON.stringify({ status:'ok', file:fname, count:data.leads?data.leads.length:0 })) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({status:'error',msg:err.message})) .setMimeType(ContentService.MimeType.JSON); } } 3. 💾 Save गर्नुस् (Ctrl+S)
4. Deploy → New Deployment click गर्नुस्
   • Type: Web App छान्नुस्
   • Execute as: Me
   • Who has access: Anyone
5. Authorize गर्नुस् (Google account allow गर्नुस्)
6. Deploy पछि आउने Web App URL तल paste गर्नुस्
